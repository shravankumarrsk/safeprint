# SafePrint

## Tech Stack
- React (Vite)
- Node.js
- Express
- MongoDB
- Tailwind CSS
- Axios
- Framer Motion
- Lucide React

## Setup
1. Install dependencies in `frontend` and `backend` folders.
2. Start MongoDB.
3. Run backend and frontend servers.

## Usage
- Frontend: `npm run dev` in `frontend`
- Backend: `npm run dev` in `backend`
